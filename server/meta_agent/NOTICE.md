# EvoAgentX Attribution

This package contains a minimal planning schema, prompt, and workflow adapter
inspired by EvoAgentX's workflow generation concepts.

Source project: https://github.com/EvoAgentX/EvoAgentX

Local reference path used during implementation:
`C:\Users\21547\Downloads\EvoAgentX-main\EvoAgentX-main`

EvoAgentX is released under the MIT License. This integration does not vendor
the full EvoAgentX runtime, optimizer, RAG modules, MCP toolkit, or dependency
tree. It adapts the lightweight planning shape of goal -> sub_tasks -> inferred
edges into ModelMirror's native workflow format.
